## 二、移动端响应式优化

### 2.1 响应式断点设置

```css
/* 超小屏幕 */
@media (max-width: 375px)

/* 小屏手机 */
@media (max-width: 480px)

/* 标准手机 */
@media (max-width: 768px)

/* 平板设备 */
@media (min-width: 769px) and (max-width: 1024px)

/* 横屏模式 */
@media (max-width: 896px) and (orientation: landscape)
```

### 2.2 移动端优化项目

#### 字体优化：
- 移动端基础字体：15px
- H1标题：32px（移动端）
- H2标题：26px（移动端）
- 行高优化：1.6-1.7
- 防止iOS自动缩放：font-size: 16px（表单元素）

#### 布局优化：
- 单列布局（768px以下）
- 两列布局（769px-1024px）
- 减少垂直间距（移动端减半）
- 优化内边距（16px标准）

#### 导航优化：
- 汉堡菜单按钮（移动端）
- 全屏菜单展开
- 触摸友好的菜单项（最小44px）
- 隐藏顶部信息栏（移动端）

#### 按钮优化：
- 全宽按钮（移动端）
- 最小触摸区域：44x44px
- 优化按钮间距
- 点击反馈效果

#### 表单优化：
- 全宽输入框
- 增大输入框高度（移动端）
- 优化表单字段间距
- 防止iOS自动缩放

#### 图片优化：

- 懒加载（Lazy Loading）

- 移动端禁用悬停效果

## 四、使用说明

### 4.1 启用优化功能

所有优化功能已自动集成到主题中，无需额外配置。主题激活后即可生效。

### 4.2 自定义配置

#### 修改企业信息：
编辑 `functions.php` 中的以下函数：
- `wens_track_seo_meta_tags()` - 修改元标签
- `wens_track_schema_markup()` - 修改结构化数据

#### 修改联系方式：
在结构化数据中更新：
```php
'telephone' => '+86-xxx-xxxx-xxxx',
'address' => array(
    'addressRegion' => '广东省',
    'addressLocality' => '具体城市'
)
```

### 4.3 推荐插件

为了进一步提升SEO效果，建议安装以下插件：

1. **Yoast SEO** 或 **Rank Math**
   - 高级SEO功能
   - XML Sitemap生成
   - 内容分析

2. **WP Rocket** 或 **W3 Total Cache**
   - 页面缓存
   - 资源压缩
   - CDN集成

3. **Smush** 或 **ShortPixel**
   - 图片压缩优化
   - WebP格式转换

4. **Contact Form 7**
   - 联系表单（已适配）
